WARP 一键脚本菜单 2025 最新版 🚀

1. 官方 WARP 安装：稳定官方客户端，支持 IPv4/IPv6 加密，适合新手入门。 
   https://pkg.cloudflareclient.com/

   首次安装 (更新公钥后)：
   wget https://pkg.cloudflareclient.com/pubkey.gpg
   sudo gpg --yes --dearmor --output /usr/share/keyrings/cloudflare-warp-archive-keyring.gpg
   echo "deb [signed-by=/usr/share/keyrings/cloudflare-warp-archive-keyring.gpg] https://pkg.cloudflareclient.com/ stable main" | sudo tee /etc/apt/sources.list.d/cloudflare-client.list
   sudo apt update && sudo apt install cloudflare-warp

   运行：
   warp-cli register && warp-cli connect


2. One-Click Bash 安装器：自动检测 Linux 发行版，一键安装 WARP + 代理配置，快速上手。 
   https://github.com/dragon-Elec/warp-installer-script

   首次：
   curl -fsSL https://raw.githubusercontent.com/dragon-Elec/warp-installer-script/main/install_warp.sh | sudo bash

   维护：
   warp-cli connect


3. 无限数据 Python 脚本：破解 WARP+ 无限流量，高级用户解锁大流量（需 Python 环境，已存档但可用）。 
   https://github.com/TheCaduceus/WARP-UNLIMITED-ADVANCED

   首次：
   git clone https://github.com/TheCaduceus/WARP-UNLIMITED-ADVANCED.git
   cd WARP-UNLIMITED-ADVANCED
   pip install -r requirements.txt

   运行：
   py warp-plus.py (先编辑 config.py 加你的 WARP Client ID)

   日志：
   cat runtime-log.txt
