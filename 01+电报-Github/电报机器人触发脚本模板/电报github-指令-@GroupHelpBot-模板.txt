<b>自定义个性化设置指令</b><br>
1️⃣ 🔹 <b>打开机器人</b>  <a href="https://t.me/GHClone4Bot">@GHClone4Bot</a><br>
2️⃣ 🔍 在私聊中，点击 <b>找到其它</b>，然后选择 <b>个性化指令</b>，点击 <b>添加</b><br>
3️⃣ ↩️ 返回到群聊，群组内发送你需要自定义的 <b>HTML 文本格式</b>，并发送。<br>
4️⃣ 💬 在回复的HTML消息后，输入命令：<code>/personal 自定义名称</code>，例如：<code>/personal github</code><br><br>
<b>如果需要删除自定义指令：</b><br>
1️⃣ 🗑️ 在群组中，直接发送命令：<code>/remove !删除指令</code><br>
2️⃣ 例如：<code>/personal !github</code><br><br>
<b>查看自己的自定义个性化指令步骤：</b><br>
1️⃣ 👤 打开机器人 <a href="https://t.me/GHClone4Bot">@GHClone4Bot</a><br>
2️⃣ 📋 找到 <b>个性化指令</b>



/personal 自定义指令名称                /remove !删除指令
